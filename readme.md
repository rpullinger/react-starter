# React Starter
My small react starter pack.

## Creating a new project
````
git clone https://github.com/rpullinger/react-starter.git projectName
rm -rf .git
git init
npm init
npm install
git add .
git commit
````

## Day to Day development
`npm run watch`

## To build
`npm run build`

## For code coverage
`npm run coverage`

## Includes
* React
* Redux
* Webpack for JS bundling
* PostCSS for CSS
* Mocha for testing
* Browser-sync for remote testing
